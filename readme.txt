Plugin para aceptar pagos de Webpay de Transbank en Chile usando WooCommerce

== Description ==
Plugin para agregar Transbank Webpay como medio de pago a WooCommerce. El usuario será guiado paso a paso en el proceso de certificación. El usuario requiere un código de comercio de Transbank para funcionar. En caso que el cliente no posea un código de comercio, lo guiaremos paso a paso en la obtención de este código

Este plugin reemplaza el anterior PayForm for WooCommerce. Aun así se mantendrá la compatibilidad con el antiguo plugin.

Se requiere de WooCommerce para que este plugin funcione.

== Installation ==
Una vez el plugin está instalado, será guiado paso a paso para configurar su tienda WooCommerce.
